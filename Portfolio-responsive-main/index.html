Recent Projects

Project 1

PIB GenMotion AI

PIB GenMotion AI automates video creation from PIB press releases using AI. Articles are summarized and converted into editable scripts by a Large Language Model. Images are sourced via the LLaMA model, scripts are translated into 13 languages, and voiceovers with captions are generated using Microsoft Text-to-Speech. Tools like MoviePy and FFmpeg add animations, producing multilingual videos ready for officer approval..
